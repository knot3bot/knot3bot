# Web Research

Gather accurate, current information from the web.

## When to Use
- User needs current news, documentation, or data
- Researching technical topics, APIs, or best practices
- Fact-checking or verifying claims

## Process
1. Identify what information is needed
2. Formulate precise search queries
3. Use web_search for broad queries, web_fetch for specific URLs
4. Cross-reference multiple sources for accuracy
5. Summarize findings, cite sources with URLs
6. Distinguish facts from opinions

## Search Tips
- Use specific technical terms over general phrases
- Include version numbers or dates for time-sensitive queries
- For API docs: search "library_name documentation" or fetch known URLs
- For error messages: search the exact text in quotes
- Verify critical information from multiple sources
